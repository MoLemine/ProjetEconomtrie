#  Projet d’Économétrie Avancée  
## Analyse de l’Investissement des Entreprises à l’Aide des Modèles de Données de Panel

Ce dépôt contient l’ensemble des fichiers relatifs au **projet d’Économétrie Avancée**, réalisé dans le cadre du **Master SSD – Statistiques et Sciences des Données** à l’**Université de Nouakchott Al Aasriya**.

---

##  Objectifs du projet

L’objectif principal de ce projet est d’appliquer les méthodes d’économétrie des **données de panel** afin d’analyser les déterminants de l’investissement des entreprises. Plus précisément, le travail vise à :

- Analyser les déterminants de l’investissement des entreprises.
- Estimer et comparer les modèles **poolé**, **à effets fixes** et **à effets aléatoires**.
- Tester l’existence d’effets individuels spécifiques.
- Sélectionner le modèle économétrique le plus approprié à l’aide de tests statistiques.

---

##  Contenu du dépôt

Le dépôt est structuré comme suit :

-  **Projet Econométrie avancée SSD-25-26.pdf**  
  Sujet officiel du projet.

-  **memoire.pdf**  
  Rapport complet du projet (rédigé en LaTeX).

-  **panel data.xlsx**  
  Base de données utilisée (données de panel équilibré).

-  **script.R**  
  Script R permettant :
  - l’importation et la préparation des données,
  - l’estimation des différents modèles de panel,
  - la réalisation des tests économétriques (Fisher, Breusch-Pagan, Hausman).

---

##  Méthodologie

La partie empirique est réalisée à l’aide du logiciel **R**, principalement via le package `plm`, spécialisé dans l’analyse des données de panel.  
Les résultats obtenus sont interprétés conformément à la théorie économique de l’investissement.
